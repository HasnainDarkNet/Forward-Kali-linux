# Download Forwarder

A lightweight Python-based file monitoring and forwarding project designed for educational purposes. The application monitors a local Downloads directory, detects newly completed files, stores a local copy, and forwards the file to a designated receiver over a TCP connection within a controlled lab environment.

## Overview

Download Forwarder demonstrates how file system monitoring and basic network communication can be implemented using Python. It is intended to help students and cybersecurity learners understand concepts such as event-driven programming, file handling, socket communication, and automation.

## Features

* Real-time Downloads folder monitoring
* Automatic detection of newly completed downloads
* Local backup of detected files
* TCP-based file forwarding
* Timestamped file organization
* Automatic retry mechanism
* Progress display during file transfer
* Structured error handling
* Lightweight and easy to understand
* Suitable for cybersecurity and Python learning labs
